# bike_sharing
共享单车出行行为分析，基于2017-05-10 ~ 2017-05-16的数据


## 使用
运行结果放在 `images/` 文件夹下  

如果要自己运行结果只需要依次运行源码( `src/py版本/` 文件夹下)从 1 - 6 即可
